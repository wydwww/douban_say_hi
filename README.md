# douban_say_hi
给新关注你的友邻发一封豆邮

发豆邮的模块来自非官方豆瓣API :https://github.com/vi5i0n/doubanrobot

基于 Python 2.7.11，需安装 beautifulsoup, lxml

命令提示符里输 pip install beautifulsoup4

lxml 安装方法请搜索

# usage
在 get_rlist.py 中修改自己的登陆邮箱和密码和发送豆邮的内容，然后运行 get_rlist.py

默认设定刷新间隔是10秒
